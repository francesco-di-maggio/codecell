# JS Objects

Shared JavaScript objects used by both Max patches.

| File | Description |
|------|-------------|
| `accel.js` | Acceleration processing |
| `clamp.js` | Value clamping |
| `energy.js` | RMS acceleration energy (leaky integrator) |
| `euler.js` | Quaternion to Euler conversion |
| `peakhold.js` | Peak-hold envelope follower with Schmitt trigger |
| `quat.js` | Quaternion utilities |
| `scale.js` | Value scaling |
